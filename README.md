# Utilitaires DAI

Le programme ClashPerfect est une copie de https://github.com/rodolphe74/ClashPerfect adapté au DAI. Cela converti une image PNG en une autre image PNG au contraintes du DAI.

Les utilitaires Python convertissent une telle image au format binaire DAI, à charger dans la mémmoire écran (soit à la fin de la mémoire).

MemView permet de visualiser un dump mémmoire d'un programme DAI.
